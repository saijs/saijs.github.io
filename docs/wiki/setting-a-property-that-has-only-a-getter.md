
# setting a property that has only a getter

----

| Language | Message                                   |
|----------|-------------------------------------------|
| 英文     | setting a property that has only a getter |

为只读的属性设置值时报出。常见于 Firefox 中。

```javascript
if(elem.tagName = "div"){
  // 注意：上面条件表达式中笔误成赋值表达式。
}
```
